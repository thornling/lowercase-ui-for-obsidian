#### lowercase ui snippet for obsidian
###### description:
a snippet that lowercases ui, not content.

###### github:
- [obsidian](https://obsidian.md) – required
- [lowercase ui snippet](https://github.com/thornling/lowercase-ui-for-obsidian) – snippet file

###### installation:
1. this snippet works with obsidian
2. clone or download this repository
3. in obsidian: go to settings > appearance > css snippets and click the folder icon to open your vault’s `.obsidian/snippets/` folder
4. move the css file into the `snippets` folder
5. back in obsidian, toggle lowercase ui on from the snippets list
